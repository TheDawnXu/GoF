# 23种设计模式示例

## 关于设计模式学习要说明的一些东西
* 设计模式并非类库
* 在类库中其实有比较多的设计模式的使用
* 设计模式的使用并非一成不变,而且多数情况下需要变通的使用

## 设计模式遵循的原则
* 对接口编程而不是对实现编程。
* 优先使用对象组合而不是继承。
